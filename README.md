<h2>Описание</h2>
Просмотрщик глобальной карты игры <b>Arcanum: Of Steamworks and Magick Obscura</b>. Подсвечивает все уникальные сектора. Может пригодиться для поиска пасхалок. Для правильной работы <b>VisorObscura.exe</b> необходимо поместить в каталог с игрой. Программа подгружает данные только из DAT-архивов (распакованные файлы не учитываются). Это альфа версия, дальнейшая разработка пока заморожена.
