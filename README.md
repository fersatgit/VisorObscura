<h2>Описание</h2>
Просмотрщик глобальной карты игры <b>Arcanum: Of Steamworks and Magick Obscura</b>. Подсвечивает все уникальные сектора. Может пригодиться для поиска пасхалок. Для правильной работы <a href=https://github.com/fersatgit/VisorObscura/releases/download/v0.1/VisorObscura.exe>VisorObscura.exe</a> необходимо поместить в каталог с игрой. Программа подгружает данные только из DAT-архивов (распакованные файлы не учитываются). Это альфа версия, дальнейшая разработка пока заморожена.
